# 大学生亚健康科普小程序

基于微信原生小程序 + 微信云开发实现，无需独立服务器，轻量化、后台零代码易更新。

## 功能清单

- 首页：快捷入口 + 推荐阅读
- 科普资讯主页面：搜索框、7 大分类标签、图文列表、短视频入口
- 图文详情页：富文本阅读、收藏、转发、关联短视频
- 短视频播放页：竖屏播放、倍速、收藏、关联图文
- 每日健康卡片：每日文案 + 海报生成 + 保存分享
- 个人中心：我的收藏、浏览历史

## 运行步骤

1. **注册小程序**：前往 [微信公众平台](https://mp.weixin.qq.com) 注册小程序，获取 AppID。

2. **开通云开发**：在微信公众平台 → 开发 → 云开发，开通并记录环境 ID。

3. **修改环境 ID**：打开 `app.js`，将 `env: 'your-env-id'` 替换为你的云开发环境 ID。

4. **导入项目**：下载安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)，选择「导入项目」，目录指向本 `health-mini` 文件夹，填入你的 AppID，勾选「不使用云服务」或「使用微信云开发」。

5. **上传并部署云函数**：
   - 在微信开发者工具中，右键 `cloudfunctions/login` → 选择「创建并部署：云端安装依赖」。
   - 等待部署完成。

6. **创建数据库集合**：在云开发控制台 → 数据库中，依次创建以下集合：
   - `articles`
   - `videos`
   - `daily_cards`
   - `favorites`
   - `history`

7. **导入示例数据**：
   - 打开 `scripts/init-db.json`，将封面、视频地址替换为你自己的云存储地址或留空使用占位图。
   - 在云开发控制台 → 数据库 → 对应集合 → 「导入」，分别导入示例数据。

8. **编译预览**：点击开发者工具「编译」，即可在模拟器中预览。

## 云 CMS 后台配置（零代码更新）

1. 打开云开发控制台 → 内容管理（CMS）。
2. 按 `docs/database.md` 中的字段创建内容模型：
   - 图文科普（articles）
   - 短视频（videos）
   - 每日健康卡片（daily_cards）
3. 为运营人员开通 CMS 操作权限，即可直接在网页后台新增、编辑、删除内容，无需写代码。

## 二期短视频接入建议

推荐方案：将视频上传至**微信视频号**，小程序中使用 `channel-video` 组件或 `video` 组件嵌入。

- 优点：不占用云存储空间，流量费用几乎为零。
- 实现：在 `videos` 集合中保存视频号 ID 或视频播放地址，页面中直接播放即可。

若使用云存储自制短视频：
- 单条控制在 3 分钟以内；
- 在云存储上传视频，获取文件 ID 或 HTTPS 地址，填入 `videos.videoUrl`。

## 避坑点

1. **云开发环境 ID**：必须替换 `app.js` 中的 `your-env-id`，否则数据库调用失败。
2. **云函数权限**：首次使用需右键上传并部署 `login` 云函数，否则无法获取 openid。
3. **数据库索引**：如果搜索变慢，记得在 `articles` 集合为 `category`、`tags`、`createTime` 创建索引。
4. **海报保存**：`wx.saveImageToPhotosAlbum` 需要用户授权，首次保存会弹窗申请。
5. **封面图**：示例数据中的封面为空，实际运营时请上传封面到云存储，并在 CMS 中填写完整 URL。
6. **订阅消息**：如需每日推送健康卡片，需在小程序后台申请订阅消息模板，并调用 `wx.requestSubscribeMessage`。
